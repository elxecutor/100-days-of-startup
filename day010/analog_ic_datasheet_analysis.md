Analog IC Datasheet Analysis — Comparators vs Op Amps

Checklist:
- **Use comparator when:** threshold detection with minimal latency and low-power standby is required.
- **Use op amp when:** linear amplification, filtering or precise waveform capture is needed.
- **Key comparator specs to check:** propagation delay, input offset, output stage type, hysteresis capability, and input common‑mode range.
- **Design notes:** add hysteresis and input filtering to prevent chatter; verify timing and noise immunity across temperature.
